# image_transformation
apply transformation functions to images
<br />
Run ui.py file.
<br />
If the number of pixels in the image is large, the calculations will take a long time.
<br />
suggested functions :
<br />a = a + 100
<br />a = a * 2
<br />a = a ** 1.6
<br />a = 255 * log(a + 1) / log(256)
<br />[0, 50] a = a ** 1.3 [50, 80] a = a ** 1.2
